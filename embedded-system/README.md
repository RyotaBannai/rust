### Qumu emulator for Cortex-M4 with Docker

- xpm: xpack manager to install qumu

### how to set up

- `docker build -t qemu .`
- run docker image above by
- check version installed `docker run --rm qemu qemu-system-gnuarmeclipse --version`

### direnv

- edit: `direnv edit .`
- `direnv allow` to allow envs
